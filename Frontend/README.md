# Veda Flutter Frontend

Flutter app scaffolded with a simple, scalable folder layout.

## Project Layout

- `lib/main.dart`: app bootstrap only
- `lib/src/app.dart`: app shell and root `MaterialApp`
- `lib/src/theme`: central theme definitions
- `lib/src/features`: feature-first modules

## Run

1. `flutter.bat pub get`
2. `flutter.bat run`

## Quality Checks

1. `flutter.bat analyze`
2. `flutter.bat test`
